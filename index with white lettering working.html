Coral Project

This project applies computer vision and machine learning to coral reef monitoring and conservation. We develop models for image segmentation, species classification, and metric visualization from underwater imagery to help track reef health and climate impacts.

Goals

  • Train ML models to recognize and classify coral species in images/videos
  • Process large datasets from reef regions (e.g., Florida, Bonaire)
  • Visualize monitoring metrics for researchers and conservation efforts

Technologies

Python, TensorFlow/PyTorch, image segmentation tools (e.g., U-Net), classification models, data processing scripts.

Looking for motivated graduate/undergraduate students to contribute—contact me if interested in joining!
